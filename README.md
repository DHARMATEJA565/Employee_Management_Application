# Employee Record Management Application

## Introduction
Welcome to the Employee Record Management Application! This Java Swing application allows users to manage employee records efficiently. With its intuitive interface, users can easily add, view, update, and delete employee records stored in a MySQL database.

## Features
- Add new employee records.
- View existing employee records.
- Update employee records.
- Delete employee records.

## Prerequisites
Before running the application, ensure you have the following installed:
- Java Development Kit (JDK)
- MySQL server
- MySQL Connector/J (JDBC driver for MySQL)
- Java Swing library

## Installation
1. Clone or download this repository to your local machine.
2. Import the project into your preferred Java IDE.
3. Set up your MySQL database and ensure it is running.
4. Run the provided SQL script (`database_setup.sql`) to create the necessary database schema and sample data.

## Usage
1. Launch the application from your Java IDE.
2. The home page will display with four buttons:
   - **Add an Employee Record**: Click to add a new employee record.
   - **View an Employee Record**: Click to view existing employee records.
   - **Update an Employee Record**: Click to update an existing employee record.
   - **Delete an Employee Record**: Click to delete an existing employee record.
3. Follow the instructions on each interface to perform the desired operation:
   - **Add**: Fill in the employee details in the provided text fields and click "ADD" and to reset the text click "RESET".
   - **View**: Enter the employee ID and click "SHOW" to view the record.
   - **Update**: Enter the employee ID and update the desired fields. Click "UPDATE" to save changes.
   - **Delete**: Enter the employee ID and click "Delete" to remove the record.
